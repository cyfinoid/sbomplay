What is SBOM?

A Software Bill of Materials (SBOM) is a formal, machine-readable inventory of software components and dependencies, used to track and manage software supply chain security and compliance.

Key benefits:

  • Identify security vulnerabilities
  • Track license compliance
  • Manage dependencies
  • Ensure supply chain security
How it Works
  1. Input Organization: Enter a GitHub organization name
  2. Discover Repositories: Find all public repositories in the organization
  3. Fetch SBOMs: Download SPDX-compliant SBOMs from GitHub's dependency graph API
  4. Analyze Dependencies: Extract and analyze common dependencies across all repositories
  5. Generate Reports: View results and export data for further analysis
