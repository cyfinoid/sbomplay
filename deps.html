Dependency Views

Choose a view format to explore your dependencies

Simple, Accessible Views

Three practical formats for viewing and analyzing your dependencies. Pick the one that works best for your workflow.

Table View RECOMMENDED

Simple searchable and sortable table like a spreadsheet

Features:
  • Search across all fields
  • Sort by any column
  • Filter by type/ecosystem
  • Export to CSV
  • See parent dependencies
Best For:

Finding specific packages, sorting by usage, getting quick answers

List View

Text-based tree view like npm ls or cargo tree

Features:
  • Collapsible tree structure
  • Clear parent-child lines
  • Copy-paste friendly
  • Like terminal output
  • Expand/collapse all
Best For:

Understanding hierarchy, seeing dependency chains, familiar format

Grid View

Advanced data grid with grouping and pivoting

Features:
  • Group by ecosystem/repo
  • Multi-column sorting
  • Inline filtering
  • Column reordering
  • Export to CSV/JSON
Best For:

Power users, data analysis, complex filtering and grouping

Quick Comparison

Getting Started
  1. Click on any view card above to open that format
  2. Select an analysis from the dropdown
  3. Use the search and filter tools to find what you need
  4. Export data if needed (CSV/JSON available)
