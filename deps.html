Available Visualization Types
Visualization Description Best For Action
Table View Sortable, filterable table showing all dependencies with detailed information including version, type, and usage count Data Analysis Export View
List View Simple hierarchical list of dependencies organized by repository Quick Overview Simple View
Tree View Hierarchical tree visualization showing dependency relationships with expandable nodes Hierarchy Relationships View
Sunburst Chart Radial visualization showing dependency hierarchy with size representing usage frequency Proportions Visual Appeal View
Force-Directed Graph Interactive physics-based graph showing dependencies as connected nodes with real-time repositioning Connections Interactive View
Radial Tree Circular tree layout radiating from center, showing dependency relationships in a compact radial format Radial Layout Compact View
Dagre Layout Directed acyclic graph with automatic hierarchical positioning showing clear dependency flow Flow Direction Clear Structure View
Grid View Card-based grid layout showing dependencies as individual cards with key information Overview Card Layout View
Visual Overview Combined visual dashboard with charts and statistics about your dependencies Statistics Dashboard View
Usage Tips
  • Use Table View for detailed analysis and filtering
  • Use List View for a quick overview
  • Use graph visualizations (Force, Dagre, Radial) to understand relationships
  • Use Sunburst to see proportional distribution
Performance Note

Graph-based visualizations (Force, Dagre, Radial, Tree) may take longer to load for projects with many dependencies. For quick access to data, use Table or List view first.
